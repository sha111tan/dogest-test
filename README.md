# Тестовое задание для CGP.APP

Небольшое mobile-first PWA для владельцев собак. Задачи по уходу за питомцем и заметки с тегами - всё работает оффлайн и догоняет сервер, когда сеть появляется снова

## Что внутри

**Список задач** — добавить, отметить задачу закрытой, удалить. Сверху кольцо прогресса и полоска: процент выполненных задач анимируется через Framer Motion 

**Заметки** — создание и редактирование в bottom sheet, теги через запятую, поиск по тексту и фильтр по тегам

**Оффлайн** — данные лежат в IndexedDB (Dexie). Без сети всё читается и пишется как обычно

**Синхронизация** — каждое изменение попадает в очередь `syncQueue`. Когда `navigator.onLine === true`, очередь уходит на `POST /api/sync`, потом подтягивается актуальное состояние с `GET /api/sync`. Конфликты решаются по `updatedAt` — побеждает более свежая версия

**PWA** — service worker кэширует статику приложения, есть manifest, можно поставить на домашний экран

## Стек

React 19, TypeScript, Vite, TailwindCSS, Dexie + dexie-react-hooks, vite-plugin-pwa (Workbox), Framer Motion

## Запуск

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

Сборка:

```bash
npm run build
npm run preview
```

Линтер: `npm run lint`

## Как проверить оффлайн

1. Открыть приложение, добавить/удалить/изменить пару задач и заметок
2. Отключить интернет
3. Данные на месте, можно редактировать дальше
4. Включить интернет - в хедере появится «Синхронизация данных...»

Mock-сервер встроен в Vite (`vite.mock-api.ts`): в dev и preview `/api/sync` хранит данные в памяти процесса. После перезапуска сервера «бэкенд» обнуляется, локальная IndexedDB - нет

## Структура проекта

```
src/
├── api/           — fetch к /api/sync
├── components/    — todos, notes, layout
├── db/            — схема Dexie
├── hooks/         — live queries + CRUD
├── sync/          — очередь и runSync
├── types/
└── utils/
```

## Что из ТЗ закрыто

| Требование | Как |
|---|---|
| Mobile-first | Вёрстка от 320px, нижняя навигация |
| IndexedDB | Dexie, три таблицы: todos, notes, syncQueue |
| Service Worker | vite-plugin-pwa + Workbox |
| Синхронизация при сети | Очередь + `online` + mock API |
| Интерактивный UI | Кольцо прогресса, layout-анимации списков, sheet для заметок |
| TypeScript | Весь src типизирован |
| iOS | safe area, dvh, PWA meta |
